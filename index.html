Format
Format
URL source + ?hs= + base64 du contenu. Le contenu est libre (Markdown, SQL, YAML, HTML, texte) et la license aussi. Le format Agent Skills est recommandé mais pas obligatoire.
Source URL + ?hs= + base64 of the content. Content is free (Markdown, SQL, YAML, HTML, text) and so is the license. The Agent Skills format is recommended but not required.
Traçabilité
Traceability
Le hash SHA-256 de l'URL source + contenu sert de trace. Chaque version peut référencer le hash de la version précédente. Recommandé, pas obligatoire.
The SHA-256 hash of source URL + content serves as a trace. Each version can reference the previous version's hash. Recommended, not required.
Signature
Signature
L'auteur peut signer le hash. Ed25519 recommandé.
The author can sign the hash. Ed25519 recommended.
Taille
Size
La taille maximum du paramètre hs dépend du navigateur :
The maximum size of the hs parameter depends on the browser:
Chrome        ~32 000 caractères / characters
Firefox       ~65 000 caractères / characters
Safari        ~80 000 caractères / characters
Edge          ~32 000 caractères / characters
Android       ~8 000 caractères / characters
La recette analyser_amendements de tricoteuses fait 4 895 octets en clair, 6 528 caractères en base64. Elle passe sur tous les navigateurs modernes. Avec compression (préfixe gz. ou br.), des skills de 250-500 Ko tiennent dans ~32 000 caractères.
The analyser_amendements recipe from tricoteuses is 4,895 bytes raw, 6,528 characters in base64. It fits in all modern browsers. With compression (prefix gz. or br.), skills of 250-500 KB fit in ~32,000 characters.
